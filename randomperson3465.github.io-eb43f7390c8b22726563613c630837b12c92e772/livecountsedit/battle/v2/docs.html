How to use Livecounts edit Battle Mode

WARNING: It is not recommended for small screens. The hotkeys features may not be available on mobile.

Set Channel Name

To set a channel name, click the old channel name and an alert message should pop up.

Set Channel Subs

To set the channel's subscriber count, type the subscriber counts into the text boxes below the counts. Then, click the "Sub Gap" text to save the values. If you cannot see the text boxes, press T on your keyboard or click the sub gap number.

Set Subscriber Gain Rate

To set the subscriber rate of a channel, click the current count and an alert message should pop up. Don't worry, if you click out you won't lose your current subscriber rate. To make the channel lose subscribers, set the rate to negative.

Hotkeys

  • 1 - Add one subscriber to the first channel
  • 2 - Add ten subscribers to the first channel
  • 3 - Add 100 subscribers to the first channel
  • 4 - Remove one subscriber from the first channel
  • 5 - Remove ten subscribers from the first channel
  • 6 - Add one subscriber to the second channel
  • 7 - Add ten subscribers to the second channel
  • 8 - Add 100 subscribers to the second channel
  • 9 - Remove one subscriber from the second channel
  • 0 - Remove ten subscribers from the second channel
  • a - Toggle abbreviated count mode (default disabled)
  • c - Clear graph (will ask for confirmation)
  • g - Toggle graph (default enabled)
  • n - Toggle negative subscriber counts (default disabled)
  • t - Toggle subscriber count text boxes (default enabled)
