How to use Livecounts edit 4.0.0

WARNING: It is not recommended for small screens. The hotkeys features may not be available on mobile.

Set Channel Name

To set the channel name, click the old channel name and an alert message should pop up.

Set Channel Subs

To set the channel's subscriber count, type the subscriber counts into the text box that says "New Subscriber Count", then click the counter. If you cannot see the text box, press H on your keyboard or click the "Live subscriber counter" subtitle.

Set Subscriber Gain Rate

To set the gain rate per hour for a channel press S on your keyboard and a prompt should show up. Follow the instructions on the prompt. To make the channel lose subscribers, set the rate to negative.

Hotkeys

  • 1 - Add one subscriber
  • 2 - Add ten subscribers
  • 3 - Add 100 subscribers
  • 4 - Add 1000 subscribers
  • 5 - Add 1 million subscribers
  • 6 - Remove one subscriber
  • 7 - Remove ten subscribers
  • 8 - Remove 100 subscribers
  • 9 - Remove 1000 subscribers
  • 0 - Remove 1 million subscribers
  • a - Audit subscribers
  • c - Clear graph (will ask for confirmation)
  • h - Toggle the "New Subscriber Count" box
  • n - Toggle negative subscriber counts (default disabled)
  • r - Reset everything to zero (will ask for confirmation
  • s - Set the subscriber rates
  • u - Toggle the count changing colo(u)rs when updating

    • URL Parameters

  • a - Audit size for the 'a' key (default: 0)
  • ab - Abbreviated mode (1 is enabled, 0 is not, default: 0)
  • an - Allow negative number counts (1 is allowed, 0 is not, default: 0)
  • c - Starting count when the page loads (default: 0)
  • g - Whether the graph should be enabled (1 is enabled, 0 is not, default: 1)
  • n - The name of the channel (default: YouTube Channel Name)
  • mr - Minimum gain rate in subs/hour (default: 0)
  • xr - Maximum gain rate in subs/hour (default: 0)
